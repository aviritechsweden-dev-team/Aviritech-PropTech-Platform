# Aviritech PropTech

![Version](https://img.shields.io/badge/version-beta-yellow)

Official repository for the Aviritech PropTech Platform.

## About

Aviritech PropTech is a modern digital real estate platform designed to simplify property discovery, tenant management, landlord management, inspections, payments, utilities, and housing accessibility.

Built for Nigeria.
Ready for Africa.

---

## Features

- Tenant Portal
- Landlord Portal
- Property Listings
- Property Verification
- Property Inspections
- Inspection Scheduling
- Secure Payments
- Flutterwave Integration
- WooCommerce Integration
- Ultimate Member Authentication
- Digital Utility Payments
- Dashboard Analytics
- Property History
- Notifications
- Admin Portal
